# Third-party assets

The app icon composites an openly-licensed pile-of-poo emoji onto a blue tile.

- **poo.png** — "Pile of Poo" from **Microsoft Fluent Emoji** (3D style).
  License: **MIT**. © Microsoft. https://github.com/microsoft/fluentui-emoji
  This is the source used by default.

- **poo-noto.svg** — "Pile of Poo" from **Google Noto Emoji**.
  License: **Apache-2.0**. https://github.com/googlefonts/noto-emoji

- **poo-twemoji.svg** — "Pile of Poo" from **Twemoji**.
  License: **CC-BY 4.0**. https://github.com/jdecked/twemoji

Select the source with the 2nd CLI arg to `iconforge` (`fluent` | `noto` | `twemoji`).
